Detex lists the paa files used in the special bis file TexHeaders.bin

Output analysis is to screen

no arguments are required. the default is texheaders.bin in the current folder.

if a filename is stated it need not be 'texheaders.bin' but currently makes no sense to do otherise

a folder can be specifed by itself. detext will look for texheaders.bin in that folder


Syntax:

DeTex [-p] // decode texheaders.bin in current directory

DeTex [-p] folder   // decode folder\texheaders.bin

Detex  [-p]  [folder\]filename.ext
