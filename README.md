# Mikero Tools

All linux Mikero Tools prepackaged in a Docker Image.
Binaries are installed in /usr/local/bin and should be accessible system wide.

## Included Tools

* convertwrp
* defxy
* dekey
* dep3d
* depac
* depew
* derap
* dertm
* detex
* dewrp
* dewss
* extractpbo
* makepbo
* moveobject
* pbodeps
* queryaddons
* rapify
